# List Tags

## ol tag

> ol: ordered list

Code:

```HTML
<ol>
    <li>Hello World 1</li>
    <li>Hello World 2</li>
    <li>Hello World 3</li>
    <li>Hello World 4</li>
</ol>
```

## ul tag

> ul: unordered list

Code:

```HTML
<ul>
    <li>Hello World 1</li>
    <li>Hello World 2</li>
    <li>Hello World 3</li>
    <li>Hello World 4</li>
</ul>
```
