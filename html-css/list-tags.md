# List Tags

## ol tag

> ol: ordered list

Code:

```HTML
<ol>
    <li>Hello World 1</li>
    <li>Hello World 2</li>
    <li>Hello World 3</li>
    <li>Hello World 4</li>
</ol>
```

Result:

```
1. Hello World 1
2. Hello World 2
3. Hello World 3
4. Hello World 4
```

## ul tag

> ul: unordered list

Code:

```HTML
<ul>
    <li>Hello World 1</li>
    <li>Hello World 2</li>
    <li>Hello World 3</li>
    <li>Hello World 4</li>
</ul>
```

Result:

```
* Hello World 1
* Hello World 2
* Hello World 3
* Hello World 4
```
